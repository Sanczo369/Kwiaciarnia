# Kwiaciarnia
Projekt Kwiaciarnia to strona internetowa dedykowana kwiaciarni. Strona nie tylko prezentuje naszą ofertę, ale także umożliwia łatwe i przyjemne zamawianie kwiatów online.
 
### Technologie
- HTML, CSS
- Python Flask
- SQLAlchemy
- Flask-Bcrypt

### Funkcje
- Przeglądanie Oferty
- Zamawianie Online
- Zdjęcia i Opisy
- Promocje i Rabaty
- Kontakt
